Pig-Min Behind 16. 괜히 해본 음모론.

mrkwang : '한국인디게임산업협회' 같은 거 생겨서, 촛불집회 하는 광장에서 인디 게임 전시회 하면 어쩌지요?
칼리토 : ...
mrkwang : 그러다가 막, 인디게임 실제 제작자들이 나타나서, '우리 게임 누구 맘대로 전시하냐!'고 항의.
칼리토 : .......
mrkwang : 전시회장에 제작은 커녕 평생 게임은 고포밖에 안해본거 같은 인상의 아저씨들이 안내한다고 서있고, 실제 안내도 안하고.
mrkwang : 이렇게 세계 최초로, 야외 광장에서 인디 게임 전시회가 벌어지는 겁니다.
mrkwang : ... 써놓고 보니 굉장히 그럴싸한 음모론.
mrkwang : 그러니까 늦기 전에, Pig-Min이 주관해서 '한국인디게임산업협회' 만들어버리는 겁니다.
칼리토 : ......
mrkwang : 아.
mrkwang : 그런거 만들어봤자, '한국인디게임제작자산업협회' 만들면 끝이구나.
