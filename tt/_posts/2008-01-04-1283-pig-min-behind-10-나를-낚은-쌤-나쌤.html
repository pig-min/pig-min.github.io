... 뭐야 이거 무서워;

이곳은 속칭 '나쌤'이라고 불리는 분의 블로그입니다. 왜 '나쌤'이나면, '나씨성을 가진 Sam'...일리 없고. 나리티(Nairrti)라는 닉을 쓰는데, 모처에서는 선생님이기 때문입니다.

뭐 그건 그렇고. 이번에 또 다른 '나쌤'의 의미가 생겼는데요. 그건 바로바로바로바로...

'나를 낚은 쌤.'


Pig-Min Behind 10. 나를 낚은 쌤, 나쌤.

사건은 대략 2주 전의 MSN 대화에서 시작되었습니다.

나쌤 : 요즘 잠을 못 자겠어요.
mrkwang : 왜요?
나쌤 : 게임 하느라고.
나쌤 : 이 게임 너무 굉장해서, 제가 1주일 반이나 해야 할거 같아요.
mrkwang : ... 1주일 반'이나'라는 표현은 좀 뭔가 이상;
나쌤 : 아니 원래 제가 게임을 빨리 하고 넘기는 편이라, 이렇게 오래 잡는거 별로 없어요.
mrkwang : 호오. 그래요? 그게 뭔데요.
나썜 : [크루세이더 킹즈(Crusader Kings)]라고, 유럽의 파라독스(Paradox Interactive)에서 만든 하드코어 게임이에요.
mrkwang : 그럼 리뷰라도 쓰시죠?
나쌤 : ... 중독 풀리고 나서.
mrkwang : ... 그러셈.

그리고 1주일 정도 더 지나니, 이 님이 리뷰를 올렸더군요.

리뷰를 보니 뭔가 굉장히 재미있고 훌륭해 보이더군요.

... 샀습니다. 일단 확장팩은 나중에나 지르려고, 홀리데이 세일하느라 14.99$로 할인한 본편부터.

mrkwang : 저도 질렀어염 질렀어염. >.<
나쌤 : 아 혹시 패치 까셨나요?
mrkwang : ... 패치도 깔아야 됨? 그런거 미리 다 해서 팔아주면 안되나.
나쌤 : 네. 그런데 패치가 이상해서, 레지스트리 픽스를 해야 해요. 인스톨 때 폴더 지정이 제대로 안되어서.
mrkwang : ... 죄송한데 한국말로; 저 개발자 아님;
나쌤 : 맞다. 확장팩 사면 패치 안 깔아도 됨.

... 그래서 확장팩도 샀습니다. 홀리데이 세일 때라서, 14.99 + 14.99 = 29.98$. 약 30$.

그리고 본편 + 확팩 다 깔고, 청운의 푸르른 꿈을 지닌 채 게임에 들어갔습니다. 우왕ㅋ굿ㅋ. 이제 유럽은 다 내꼬얌. 다 먹어 치울테당~

... 30초 후 나옴. 이거 뭐 날 보고 어쩌라는거야! 최대 양보해서 [삼국지] 정도 상상했는데, 이건 뭐 아주 작은 글씨의 영어로 화면이 가득한데, 딱히 튜토리얼도 없이 설명 페이지 1개 던져준 후 곧바로 게임에 돌입! 게다가 실시간! 대체 나한테 뭘 바라는거야아아아앙!

다시 MSN에서 띠딩.

mrkwang : ... 이거 뭐죠? 어떻게 하라는것?
나쌤 : ... 혹시 이런 하드코어 게임 안해보셨음?
mrkwang : ... 니마 매너염; 그런걸 했을리가 없잖;
나쌤 : ...

이렇게 웅이 아범 광님은 악의 세력 나쌤에게 낚이고.

판도라 TV 파라독스 인터랙티브는 해가 떠오르는 동쪽 나라의 거금 30$를 겟했습니다.

... 환불해줘 ;ㅁ;

P.S. : 앞으로 '나쌤'이 '하드코어 게임'이라고 부르는 것은, 절대 옆에도 안갈테다.
