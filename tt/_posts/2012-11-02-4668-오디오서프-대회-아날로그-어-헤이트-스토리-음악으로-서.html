* [아날로그 : 어 헤이트 스토리]는 매우 훌륭한 비주얼 노벨로써, 전문 번역가 김지원씨에 의해 완벽 한국어화(한글화) 되었습니다.

1. 참가방식
- 아래 올려둔 지정 곡을 다운받고 기간 내에 서핑해서 순위권에 드시면 됩니다.
- 종료 직후 운영진이 차트의 스크린 캡쳐를 찍습니다. 즉 스크린 캡쳐 이후에 등록된 기록은 무효입니다.

2. 참가기간
2012/11/11(일) 23:59까지.

3. 참가 신청 / 지정곡
- 지정곡 (링크에 마우스 우클릭해서 다운) : [아날로그 : 어 헤이트 스토리] OST의 2번 곡 Hyun-Ae(Innocence)
- 경기를 위해 MP3 곡의 태그를 바꿔두었습니다. 이미 이 곡을 갖고계시더라도 무조건 여기서 다시 받아서 하셔야 합니다. 해당 곡의 차트는 여기.
- 참가하신 분들은 본인 사전 확인을 위해 보고계신 이 공지의 리플에 다음 사항들을 적어주시길 부탁드립니다. (공개글 / 비밀글 모두 가능.)
 1) 오디오서프 ID
 2) 제리얼넷 ID (제리얼넷 회원가입)

4. 상품
* Elite 부문
1위 (1명) : 인디 로얄 겟어웨이 번들 (아날로그 : 어 헤이트 스토리 포함) 1개.
2-3위 (2명) [아날로그 : 어 헤이트 스토리] 스팀 키 각각 1개.
* Pro 부문
1위 (1명) : 인디 로얄 겟어웨이 번들 (아날로그 : 어 헤이트 스토리 포함) 1개.
2-3위 (2명) [아날로그 : 어 헤이트 스토리] 스팀 키 각각 1개.

5. 선정방법
- 본 행사를 준비한  [pig-min] mrkwang / [DC]Zerial-KAISTer은 수상에서 제외됩니다.
- 스크린 캡쳐를 기준으로 각 부문별의 차트 순위별로 선정합니다.

6. 기타
- 상품 수령을 위해 제리얼넷에 무조건 가입해주셔야 합니다. 상품은 제리얼넷 계정을 통해 지급됩니다. 제리얼넷 ID가 오디오서프의 ID와 같을 필요는 없습니다.
- 참가때 본인 확인용 리플 달아주시는 것 잊지 마시길. 본인 확인용 리플이 없으면 수상에서 무조건 제외됩니다.
- 주최측 사정상 해당 상품의 증정이 불가능할 시 수상자와의 협의하에 다른 상품으로 증정할 수도 있습니다.
- 당연히(!) 완전 다른 곡에 태그만 붙여 오시는 경우는 수상 박탈합니다. 오디오서프 차트에 곡 그래프로 판정합니다.
- 어차피 친구에게 전도하려면 사서 선물해야 하니, 그냥 상품으로 타서 줍시다.
